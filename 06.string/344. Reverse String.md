# Code
```cpp []
class Solution {
public:
    void reverseString(vector<char>& s) {
        long long a = 0;
        long long b = s.size()-1;
        while(a <= b){
            char tmp = s[a];
            s[a] = s[b];
            s[b] = tmp;
            a++;
            b--;
        }
    }
};
```
만약 파이썬으로 풀 경우 s = s[::-1] 하면 안된다  
s[:] = s[::-1] 이렇게 해야 함  
전자는 지역변수 s 이름을 새 리스트에 다시 바인딩하는 것이므로 호출한쪽에서 가지고 있는 원래 리스트는 안바뀌어서 틀린다. 후자는 원래 리스트의 전체 슬라이스에 새 내용 덮어쓰기라서 원본 리스트 객체 자체가 뒤집혀서 맞는 풀이
