# Intuition
<!-- Describe your first thoughts on how to solve this problem. -->
문장을 주면 대소문자 구분하지 않고 마침표와 구두점 등 문장부호도 구분하지 않고, banned 에 없는 단어중 가장 큰것을 내보내야 한다

# Approach
<!-- Describe your approach to solving the problem. -->
문장부호 걸러내는건 정규식으로 하고, Counter사용

# Complexity
- Time complexity:
<!-- Add your time complexity here, e.g. $$O(n)$$ -->
O(n)... 이 아니고  
N 이 paragraph의 길이(또는 단어의 수)  
M 이 banned 리스트의 단어 수일때  
- re.sub와 split에서는 O(N)
- if word not in banned 에서 banned가 리스트이므로 in 연산이 O(M), 그래서 이 부분이 O(N*M) 이 된다
- Counter, most_common은 O(n)

- Space complexity:
<!-- Add your space complexity here, e.g. $$O(n)$$ -->
O(n)

# Code
```python3 []
class Solution:
    def mostCommonWord(self, paragraph: str, banned: List[str]) -> str:
        words = [word for word in re.sub(r'[^\w]',' ',paragraph).lower().split() if word not in banned]
        counts = collections.Counter(words)
        return counts.most_common(1)[0][0]
```
